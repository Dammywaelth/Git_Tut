Basic Commands

$ git init (initializes a git repository)

$ git branch -m master main (changes your mi=ain branch from master to main)

$ git config --global user.name "your name"

$ git config -1 (list all the details you submitted after git int)

$ git config --global user.email "your email"

$ git add filename (adds a file to the staging area)

$ git add . (adds all files to the staging area)

$ git rm --cached . -r (removes all files from the staging area)

$ git commit -m "your custome message" (saves files in the staging area to git)

$ git status (shows the status of the working tree)

$ git clone (clones an existing repository)

$ git pull (pull the latest changes from the remote repository to your local machine)

$ git push (pushs your local repository to your remote repository)

Other commands

$ git log (gives you a log of your commit history. Each commit history has a hash number)

$ git show hashNumber (shows you what is inside that particular commit)

$ git diff (gives you the difference between the current updated file and the previous commit)
